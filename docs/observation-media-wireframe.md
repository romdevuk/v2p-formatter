## 6. Visual Design Specifications

### 4.1 Color Scheme
- **Overall Theme**: Dark theme throughout
- **Background**: `#1e1e1e` (dark)
- **Panels**: `#2a2a2a` (darker gray)
- **Borders**: `#555` (medium gray)
- **Text**: `#e0e0e0` (light gray)
- **Text Editor**: Dark background (`#1e1e1e`), light text (`#e0e0e0`)
- **Preview**: Dark background (`#1e1e1e`), light text (`#e0e0e0`)
- **Placeholder Colors**: Rainbow color coding (each placeholder gets unique color)
  - Color 1: `#ff6b6b` (red)
  - Color 2: `#4ecdc4` (cyan)
  - Color 3: `#45b7d1` (blue)
  - Color 4: `#f9ca24` (yellow)
  - Color 5: `#6c5ce7` (purple)
  - Color 6: `#a29bfe` (lavender)
  - Color 7: `#fd79a8` (pink)
  - Color 8: `#00b894` (green)
  - Cycles through colors for additional placeholders
- **Selected Media**: `#667eea` (purple border)
- **Status Colors**:
  - âœ… Assigned: `#51cf66` (green)
  - âš ï¸ Partial: `#ffd43b` (yellow)
  - âŒ Unassigned: `#ff6b6b` (red)

### 4.2 Typography
- **Headers**: 18px, bold, `#e0e0e0`
- **Body Text**: 14px, regular, `#e0e0e0`
- **Placeholder Text**: 14px, monospace, highlighted
- **Labels**: 12px, `#999`
- **Code/Placeholders**: Monospace font

### 4.3 Spacing
- **Panel Padding**: 20px
- **Component Margin**: 15px
- **Thumbnail Gap**: 10px
- **Button Padding**: 10px 20px

### 4.4 Interactive Elements

#### Thumbnails
- **Size**: 120x90px (images), 120x90px (videos)
- **Hover**: Scale 1.05, border highlight
- **Selected**: Purple border, checkmark overlay
- **Dragging**: Opacity 0.5, cursor changes

#### Placeholder Cards
- **Collapsed**: Show name and status only
- **Expanded**: Show media list and controls
- **Hover**: Background lightens slightly
- **Active Drop Zone**: Highlighted border, pulsing animation

#### Buttons
- **Primary**: Purple background (`#667eea`), white text
- **Secondary**: Dark gray background, light text, border
- **Danger**: Red background for remove actions
- **Hover**: Slightly lighter background

## 8. Accessibility Features

1. **Font Size Toggle**:
   - Quick access button: [Aa] Regular / [AA] Big
   - Located in header/toolbar
   - Applies to all text in interface (editor, preview, labels)
   - Toggle between regular and big font sizes
   - Persists user preference

2. **Keyboard Navigation**:
   - Tab through all interactive elements
   - Enter/Space to activate
   - Arrow keys for media selection
   - Escape to close/cancel

3. **Screen Reader Support**:
   - ARIA labels on all buttons
   - Descriptive alt text for thumbnails
   - Live regions for status updates

4. **Visual Feedback**:
   - Clear focus indicators
   - High contrast text (dark theme with light text)
   - Loading states for async operations
